计算机网络

发表于 2024-09-17

理论

现在许多关于计算机网络的书都是从就爱三级网络的架构出发，分出几个层次进行详细介绍，更多需要读者自行联系贯通，故本文放弃这种传统的讲法，以主线贯通带出知识，一目了然。

从计算机开始

我们手中的计算机，包括类似的信息设备，如：手机，智能手表等，都处于网络边缘之上，统称为主机或端系统。

这里我们只以计算机为例，在计算机上，我们拥有许多奇形怪状，五颜六色的应用程序，这些应用程序在计算机上运行，我们就按操作系统的说法，称之为进程。有些进程与进程之间是需要相互联系的，在你的计算机上，你的加速器可能需要获取游戏的信息；在你的计算机和别人的计算机上，可能两个微信正在相互交换数据。这是如何做到的呢？

首先，进程会根据开发者规定的应用层协议，对数据进行打包，形成报文，然后把报文投递到套接字接口，之后运输层会根据开发者规定的运输层协议对报文进行分组交换。

应用层协议(application-layer protocol)
1.交换的报文类型
2.各种报文类型的语法
3.字段的语义
4.确定一个进程何时以及如何发送报文，对报文进行响应的规则

万维网（Web, World Wide Web）
Web的应用层协议是超文本传输协议（HyperText Transfer Protocol，HTTP）

Web页面是由对象组成的，一个对象只是一个文件

class Solution 
{
public:
    using pii=pair<int, int>;
    #定义别名pii，指向pair<int, int>类型
    vector<int> getFinalState(vector<int>& nums,int k,int multiplier)
    {   
        vector<int> res(num.size());
        #vector动态分配内存
        #设置初始值，有利与减少分配操作，但是如果初始值设置过大，会浪费内存空间。
        #分配操作过多可能影响内存的连续性

        priority_queue<pii,vector<pii>,greater<>> pq;
        for(int i=0;i<nums.size();i++)
        {
            pq.emplace(nums[i],i);

        }

        while(k--)
        {
            auto t=pq.top();
            pq.pop();
            t.first *= multiplier;
            pq.push(t);

        }
        // 将队列中的元素存储到结果数组中
        while (!pq.empty()) {
            auto [val, pos] = pq.top();
            pq.pop();
            res[pos] = val;
        }
        return res;
    }
};
